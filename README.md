# Task Management System (Spring Boot)

This is a Spring Boot REST API project for managing tasks.

The system supports creating users, categories, priorities, and tasks.  
Each task is linked to a user, category, and priority, and includes a status.

## Features
- User CRUD operations
- Category CRUD operations
- Priority CRUD operations
- Task CRUD operations
- Update Task Status (PENDING, IN_PROGRESS, DONE)
- Global exception handling
- Swagger OpenAPI integration

## Technologies Used
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 / MySQL
- Lombok
